## Hooks API

| Method   | URI                | DESCRIPTION       |
| - | - | - |
| GET      | /api/v1/hooks/     | list hookss       |
| POST	   | /api/v1/hooks/     | create a hook     |
| GET      | /api/v1/hooks/:id/ | get a hook        |
| PUT      | /api/v1/hooks/:id/ | update a hook     |
| DELETE   | /api/v1/hooks/:id/ | delete a hook     |
